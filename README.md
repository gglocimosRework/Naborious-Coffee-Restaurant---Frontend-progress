# Naborious Coffee Restaurant(fictional versions)

these websites are about a fictional coffee & brunch restaurant, one was created using HTML and CSS, while another one is generated by JS
for responsiveness and interactivity.  
The goal is to demonstrate practical frontend skills, project organization, and the ability to improve a web application step by step.
