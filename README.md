# Naborious Coffee Restaurant(fictional versions)

This project showcases a fictional coffee & brunch restaurant website. It includes two versions:
- *A version built with HTML and CSS* 
- *An interactive version enhanced with JavaScript(for responsiveness and interactivity)*

The goal is to demonstrate practical frontend skills, project organization, and the ability to improve a web application step by step.

## How to view
### 1. Clone the repository:

- git clone https://github.com/gglocimosRework/Naborious-Coffee-Restaurant---Frontend-progress.git

 ### 2. Open index.html in your browser or view the live site here to see the version:
- [Github Pages](https://gglocimosrework.github.io/Naborious-Coffee-Restaurant---Frontend-progress/)
